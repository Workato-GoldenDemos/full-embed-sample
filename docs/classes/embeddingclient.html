on

  • on<TMessageType>(messageType: TMessageType, handler: MessageHandler<(EmbeddingSourceMessageOfType<TMessageType, EmbeddingClickMessage> | EmbeddingSourceMessageOfType<TMessageType, EmbeddingNavigatedMessage> | ... 4 more ... | EmbeddingSourceMessageOfType<...>)["payload"]>): this
  • on(messageType: "*", handler: MessageHandler<EmbeddingWorkatoMessage>): this

  • Allows to subscribe to messages of specific type, emitted by embedded Workato app.

    See EmbeddingWorkatoMessage for the complete list of emitted messages.

    Example:

    Workato.on('navigated', payload => {
  console.log(`Workato navigated to ${payload.url}`);

  if (payload.replaced) {
    console.log('And URL was replaced in browser history');
  }
});

    Type parameters

    • TMessageType: "click" | "navigated" | "loaded" | "unloaded" | "heightChange" | "error" | "connectionStatusChange"

    Parameters

    • messageType: TMessageType

      Value of the type property of EmbeddingWorkatoMessage (e.g. navigated).

    • handler: MessageHandler<(EmbeddingSourceMessageOfType<TMessageType, EmbeddingClickMessage> | EmbeddingSourceMessageOfType<TMessageType, EmbeddingNavigatedMessage> | ... 4 more ... | EmbeddingSourceMessageOfType<...>)["payload"]>

      Will be called with message's payload as the first parameter.

    Returns this

handleNavigation

  • handleNavigation(config?: NavigationConfig): void

  • Helper method that allows to synchronize current Workato URL with current vendor's URL.

    It does the following:

    1. Calls NavigationConfig.onWorkatoNavigation every time after embedded Workato app is navigated to another page (Workato URL changes). It's also called right after initialization of Workato app.

      If this handler is omitted or event.preventVendorUrlChange is not called, current browser's URL will be updated to corresponding embedding URL using History API.

    2. Calls NavigationConfig.onVendorNavigation every time user clicks on embedding links in the vendor's webapp or clicks browser's back/forward buttons.

      If this handler is omitted or event.preventWorkatoUrlChange is not called, navigateTo method will be called with the corresponding Workato URL.

      Also if this handler is omitted or event.preventVendorUrlChange is not called, current browser's URL will be updated to corresponding embedding URL using History API.

    These handlers can be used, for example, to highlight currently active embedding link in the vendor's webapp or to properly embed Workato into an existing SPA app with it's own routing solution.

    Parameters

    • Default value config: NavigationConfig = {}

    Returns void

Helper Methods

constructEmbeddingUrl

  • constructEmbeddingUrl(workatoUrl: string): string

  • Constructs embedding URL out of Workato URL.

    Example:

    Workato.configure({embeddingUrlPrefix: '/integration'});
Workato.constructEmbeddingUrl('/') // => '/integration/'
Workato.constructEmbeddingUrl('/recipes') // => '/integration/recipes'
Workato.constructEmbeddingUrl('/recipes/1?a=1#hash') // => '/integration/recipes/1?a=1#hash'

    Parameters

    • workatoUrl: string

    Returns string

extractWorkatoUrl

  • extractWorkatoUrl(vendorUrl: string): string | null

  • Extracts Workato URL from vendor URL.

    Returns null if vendor URL is not embedding URL.

    Example:

    Workato.configure({embeddingUrlPrefix: '/integration'});
Workato.extractWorkatoUrl('/integration') // => '/'
Workato.extractWorkatoUrl('/integration/bar?a=1#hash') // => '/bar?a=1#hash'
Workato.extractWorkatoUrl('https://vendor.com/integration/bar?a=1#hash') // => '/bar?a=1#hash'
Workato.extractWorkatoUrl('/blah') // => null
Workato.extractWorkatoUrl('https://google.com/foo/bar?a=1#hash') // => null

    Parameters

    • vendorUrl: string

      Vendor URL. May or may not contain an origin.

    Returns string | null

generateIFrameUrl

  • generateIFrameUrl(token: string, workatoUrl?: string): string

  • Generates a URL that can be used as a value for embedding iframe's src attribute.

    Parameters

    • token: string

      JWT token for authorization

    • Optional workatoUrl: string

      Specifies Workato URL that will will be initially opened in the iframe. If omitted, Workato URL will be extracted from the current browser's URL and if it's not an embedding URL, an error will be thrown.

    Returns string
